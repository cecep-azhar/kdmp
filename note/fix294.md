### FITUR #8: Denda Keterlambatan Angsuran
**Status:** ✅ FIXED
**File:** `src/collections/Loans/index.ts`, `src/globals/Settings.ts`

**Perbaikan:**
1. Added `latePenaltyPerDay` setting in Settings (Konfigurasi Pinjaman tab)
   - Default: 0.5% per hari

2. Implemented automatic penalty calculation in Loans beforeChange hook:
   - Checks each installment's due date against current date
   - If overdue, marks installment as 'overdue' status
   - Calculates penalty: `installment.total * penaltyRate * daysOverdue`

3. Penalty rate is configurable via Admin → Settings → Konfigurasi Pinjaman

**Formula:**
```
Denda = Total Angsuran × (Denda%/hari) × Jumlah Hari Terlambat
```

**Catatan:** Denda dihitung secara otomatis saat loan di-load/update. Untuk penagihan aktual, perlu dilakukan secara manual oleh staff.
